# 关于学分

大一的基础课程很重要。如果分数高了，之后干很多事情就比较容易。

# 关于学习

没有必要对每个定理的细节了如指掌。只需要有一定印象即可。关键还是掌握**核心思维**。

对于定理，如果经常运用，那么肯定可以比较熟练；如果运用得少导致没那么熟练，那就说明它没那么重要。

## 关于学习的方向

大一春学期，可以学一下：

1. 算法
    1. 尤其可以多多关注**排序**的部分
2. 概率论
    1. EECS 70 就足够了

# 关于科研

大一转大二暑假可以开始接触。之前还是以学习、打基础为主。

大一可以看看一些以**宏观性**介绍为主的论文，对领域的 Bigger Picture 有一些了解。
